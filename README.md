# Storm Update

Файл для проверки обновлений приложения Storm.

## Как обновить версию

1. Собрать новый APK
2. Загрузить `Storm.apk` в этот репозиторий (заменить файл)
3. Обновить `version.json` — изменить `version`, `versionCode`, `changelog`
4. Если обновление обязательное — поставить `"required": true`

## version.json формат

| Поле | Описание |
|------|----------|
| `version` | Версия строкой (1.1, 1.2, ...) |
| `versionCode` | Числовой код (11, 12, ...) — сравнение по нему |
| `url` | Ссылка на APK файл |
| `changelog` | Что нового |
| `required` | true = заставить обновиться |
